## 📌 Abstract
A project entitled "Website for Wonderla" provides
services where users can search and view information
about this amusement park online — without physically
visiting the park for enquiries.

---

## 🎯 Problem Statement
Before this system, visitors had to physically visit
Wonderla or wait in long queues to get ticket and
park information. This system solves that by providing
all information online.

---

## ✅ Goals
- Reduce paper work
- Save time by booking tickets online
- Provide clear information about the park
- Avoid errors in manual processes
- Improve management of updated information
